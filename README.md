Laporan Praktikum Keamanan Jaringan

Simulasi Serangan SYN Flood (DoS) dan Mitigasi Firewall

Repositori ini berisi dokumentasi lengkap mengenai simulasi serangan Denial of Service (DoS) menggunakan metode SYN Flood pada aplikasi target DVWA, serta langkah-langkah mitigasinya menggunakan IPTables.
Deskripsi Tugas

    Nama: Muh. Tazkiyah Islam Kamaluddin

    Topik: Keamanan Jaringan - SYN Flood Attack & Mitigation

    Target Environment: Damn Vulnerable Web Application (DVWA) pada Server Apache

Isi Repositori

    Laporan PDF: Berisi penjelasan detail, langkah-langkah kerja, gambar pendukung, dan analisis hasil praktikum.

    Dokumentasi Visual: Gambar-gambar hasil monitoring resource sistem (CPU usage) sebelum dan sesudah mitigasi.
